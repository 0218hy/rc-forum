# Frontend

# Starting Frontend
1. Install Dependencies
    ```npm install```
2. Start the development server
    ```npm run dev```


# Theme colour
Maroon: #800000 
Light Pink: #f9e8ea

The colours are chosen to refelct CAPT colours.
MUI components are used for layout, card, buttons and accordions.

# Layout
The layout adapts to different screen sizes using MUI's Box and Typography componenets.
